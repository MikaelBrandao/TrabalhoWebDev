Trabalho Web Dev

Integrantes: Guilherme Telles, Mikael Brandão, Matheus Fernandes

Sistema de gerenciamento de livros desenvolvido para a disciplina de Desenvolvimento Web. Permite cadastrar e visualizar livros por meio de uma interface web integrada a uma API.

Funcionalidades

- Cadastro de livros
- Listagem de livros cadastrados
- Comunicação entre Front-end e Back-end
- Armazenamento temporário em memória

Tecnologias Utilizadas
- HTML
- CSS
- JavaScript
- TypeScript
- Node.js
- Express
- Git
- GitHub

Como Executar
1. Instale as dependências:

```bash
npm install
```

2. Compile o TypeScript:

```bash
npm run build
```

3. Inicie o servidor:

```bash
npm start
```

4. Acesse no navegador:

```text
http://localhost:3000
```

Observações

Os dados são armazenados apenas em memória e são perdidos ao reiniciar o servidor.
O código foi desenvolvido de forma colaborativa pelos integrantes utilizando GitHub e Visual Studio Code. O ChatGPT foi utilizado, principalmente, como ferramenta de apoio na criação de commits e descrições do projeto.
Para a base do projeto foi utilizado o material de apoio do Professor Edgard da Cunha Pontes.
